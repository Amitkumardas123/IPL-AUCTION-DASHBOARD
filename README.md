# 🏏 IPL Auction Dashboard – Google Sheets + Apps Script

An interactive IPL-style auction simulation dashboard built using Google Sheets and Google Apps Script.  
This project automates live player auctions, team bidding, budget tracking, and real-time auction analytics inside a spreadsheet environment.

The dashboard transforms Google Sheets into a dynamic auction management system with automated workflows, interactive UI behavior, and real-time updates.

---

# 🚀 Key Features

- 🧠 Real-time IPL auction simulation
- 💰 Automatic team budget management
- 🎲 Random player generation system
- 📊 Live sold/unsold player tracking
- 🏷️ Indian & Overseas player category handling
- ⚡ Dynamic bidding system with validation
- 📋 Automated selling price updates
- 🔄 Real-time sheet synchronization
- 🖥️ Interactive spreadsheet-based dashboard
- 🔔 Custom toast notifications and popups

---

# 📌 Project Overview

This project simulates the workflow of an IPL Mega Auction where multiple teams compete to bid on players while managing their budgets strategically.

The system automatically:
- Picks random unsold players
- Handles bid increments
- Validates team balances
- Restricts overseas player limits
- Updates sold player records
- Tracks remaining players
- Displays live auction progress

The complete auction process is managed using Google Apps Script automation integrated with Google Sheets UI components.

# 🎮 How the System Works

## 1️⃣ Random Player Selection
A random unsold player is automatically selected from the database.

## 2️⃣ Live Bidding
Teams place bids through dedicated controls/buttons.

## 3️⃣ Validation System
The system checks:
- Team balance
- Bid increments
- Overseas player limits

## 4️⃣ Player Sold Process
After a successful bid:
- Team budget updates
- Selling price updates
- Player status changes to Sold
- Sold player count updates automatically

## 5️⃣ Live Auction Analytics
Dashboard displays:
- Sold players
- Remaining players
- Team-wise purchases
- Current auction status

# 🔧 Setup Instructions

## Step 1
Clone this repository:

```bash
git clone https://github.com/yourusername/ipl-auction-dashboard.git
```

## Step 2
Open Google Sheets.

## Step 3
Go to:

```bash
Extensions → Apps Script
```

## Step 4
Paste all `.gs` files into the Apps Script editor.

## Step 5
Connect the script with your spreadsheet.

## Step 6
Run the project and authorize permissions on first launch.
